Client onboarding

Slave installer front door

This page is the simplest secure entry point: it collects the basic client details in-browser, then sends the client to a private tar bundle or repo release asset. If the download works, repo access is working too.

Expected flow

  • Open this page from GitHub Pages.
  • Fill the basic client details.
  • Download the tar bundle from the private link.
  • Run the local installer from the unpacked bundle.
  • Complete the onboarding health check after install.
